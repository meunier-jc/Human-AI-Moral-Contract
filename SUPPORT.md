# Obtenir de l'aide pour le projet CIP

Nous sommes ravis que vous vous intéressiez au Collaborative Integrity Pact (CIP) ! Voici comment vous pouvez obtenir de l'aide ou poser des questions concernant le projet.

## 1. Questions Générales et Discussions

Pour les questions générales, les discussions sur les principes du CIP, les idées de nouvelles fonctionnalités, ou pour partager vos expériences d'utilisation du pacte, veuillez utiliser la section **Discussions** de GitHub.

[Accéder aux Discussions GitHub](https://github.com/meunier-jc/Human-AI-Moral-Contract/discussions)

C'est le meilleur endroit pour :

*   Poser des questions sur la philosophie ou l'application du CIP.
*   Partager des cas d'utilisation ou des retours d'expérience.
*   Proposer des améliorations conceptuelles.
*   Interagir avec la communauté.

## 2. Rapports de Bugs et Demandes de Fonctionnalités

Si vous avez identifié un problème (bug) dans la documentation, une incohérence, ou si vous souhaitez suggérer une fonctionnalité spécifique pour le cadre, veuillez ouvrir une **Issue** sur GitHub.

[Ouvrir une nouvelle Issue](https://github.com/meunier-jc/Human-AI-Moral-Contract/issues/new/choose)

Avant d'ouvrir une issue, veuillez :

*   Vérifier les [issues existantes](https://github.com/meunier-jc/Human-AI-Moral-Contract/issues) pour voir si votre problème ou suggestion a déjà été abordé.
*   Utiliser les modèles d'issues disponibles pour nous fournir toutes les informations nécessaires.

## 3. Contributions au Code ou à la Documentation

Si vous souhaitez soumettre des modifications directes au projet (corrections de fautes de frappe, améliorations de la clarté, ajout de sections, etc.), veuillez créer une **Pull Request (PR)**.

[Créer une nouvelle Pull Request](https://github.com/meunier-jc/Human-AI-Moral-Contract/compare)

Avant de soumettre une PR, veuillez :

*   Lire notre [guide de contribution](CONTRIBUTING.md).
*   Utiliser le modèle de Pull Request pour décrire vos changements.
*   Vous assurer que vos modifications sont conformes à notre [Code de Conduite](CODE_OF_CONDUCT.md).
